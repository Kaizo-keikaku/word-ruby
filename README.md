# Wordルビ自動振りツール

Word文書(.docx)内の特定の単語に自動でルビ（ふりがな）を振るWebアプリケーションです。

## 機能
- **ファイルアップロード**: .docx ファイルをドラッグ＆ドロップでアップロード。
- **ルビ設定**: 対象の単語とルビをリスト形式で設定可能。CSVインポートも対応。
- **適用モード**:
  - `once`: 文書全体で最初の1回のみ
  - `per_page`: ページごとに最初の1回のみ
  - `all`: すべての該当箇所
- **ダウンロード**: 変換後のファイルを即座にダウンロード。

## 技術スタック
- Python 3.12+
- Streamlit
- python-docx
- lxml
- pandas

## ローカルでの実行方法

1. 依存ライブラリのインストール
```bash
pip install -r requirements.txt
```

2. アプリケーションの起動
```bash
streamlit run app.py
```

3. ブラウザでアクセス
通常は `http://localhost:8501` で起動します。
